# Requirements Definition
Aim

The solution aims to deliver instructional content through an intuitive web browser interface that responds transparently provides clear directions based on user inputs.
### Functional Requirements
Incorporate an interactive UI:
Problem: To build a website that functions properly

### State Driven Navigation: The UI must maintain an internal state to allow users to toggle between distinct instructional modules and sections of content.
Input Processing and Validation
System must capture user inputs for relevant and logic based responses
Dynamic View Display
UI must update in real time to show progress, status or feedback changes without manual page refreshes
### Data Storage

System must use a data structure to store and organise information
# Non Functional Requirements

### Performance (Efficiency)
System must update within 100ms of user interaction for a smooth experience
Usability
Interface must incorporate a 'three click rule' where users are able to find educational content within three clicks
### Data Encryption
System must feature secure data encryption to a recognised standard
### Usability(UX)
Browser Interface must function responsively on a variety of desktop and mobile devices
# Constraints
### Resource Limitation
Must use free and open source tools not purchased or premium services
Product must be accessible through a standard web browser without the need of paid, specialised tools 
### Technology
Development is limited to HTLM5, CSS3, Python and Javascript
Completion Date
Product must be complete by Term 2 Week 2
### Data Storage
System must posses internal data structures(Arrays or Dictionaries) for data handling
# Acceptable Criteria
### Performance
Interface is able to accurately respond to user inputs within 100ms for smooth experience
### Usability
All instructional content must be reached in a maximum of three inputs and clicks from the main dashboard
### Navigation
Users must be able to navigate between various modules without any incomplete or broken page reloads or links
